Compiling h5Decom 

Dependencies


-gcc>=7
-libhdf5
-boost>=1.58 (built statically)
-Qt>=5.7
-cmake



For Linux


Clone latest:



git clone https://github.com/gabegordon/CXXDecomQt 



Create build directory:



cd CXXDecomQt && mkdir build && cd build 



Generate Makefile:



cmake -DCMAKE_BUILD_TYPE=Release .. 



And Build:



cmake --build . 





For Windows


First download and install MSYS2 from:
http://www.msys2.org
Run MSYS2 and get upgrades:



pacman -Syu 



Obtain Pre-Built Qt binary files:



pacman -S mingw-w64-x86_64-qt5 



Open the following file:



<msys2 install dir>\mingw64\include\c++\7.1.0\x86_64-w64-mingw32\bits\c++config.h 



And comment out:


_GLIBCXX_HAS_GTHREADS 1 



Clone latest:



git clone https://github.com/gabegordon/CXXDecomQt 



Create build directory:



cd CXXDecomQt && mkdir build && cd build 



Ensure mingw64 is in PATH and run CMake:



cmake -G"MinGW Makefiles" -DCMAKE_BUILD_TYPE=Release .. 



Build:



cmake --build . 



Copy deps folder contents into the executable directory to run.
